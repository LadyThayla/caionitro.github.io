GIT - Unicesumar Caio Mariano

Instalação:

Baixe o GIT para a sua plataforma (Win,Linux ou OSX)

Criando um novo repositório


Crie uma nova pasta, abra o terminal e digite git init para criar um novo repositório.

Clonando um repositório


Para clonar (copiar) um repositório LOCAL execute o comando no terminal

git clone /way/to/repo

Clonando um repositório


Para clonar um repositório REMOTO execute o comando no terminal

git clone user@server:/way/to/repo

Fluxo de tabalho

  • Seus repositórios locais consistem em três "árvores" mantidas pelo git.
  • A primeira delas é sua Working Directory que contém os arquivos vigentes.
  • A segunda Index que funciona como uma área temporária e finalmente a HEAD que aponta para o último commit (confirmação) que você fez.

Adicionando e Commitando

Após as alterações no arquivo, digite git add *arquivo* ou git add . para adicionar todos os arquvos modificados para a INDEX, onde é o primeiro passo no fluxo.

Adicionando e Commitando

Para confirmar as mudanças (fazer um commit), use git commit -m "mensagem"
