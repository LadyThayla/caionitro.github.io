Oque é o Vagrant?

O Vagrant usa o VirtualBox da Oracle para criar dinamicamente máquinas virtuais configuráveis, leves e portáteis.

Benefícios de Usar o Vagrant
  • Todos os membros de uma equipe idealmente têm ambientes de desenvolvimento idênticos: mesmas dependências, mesmas versões, mesmas configurações, mesmo tudo.
  • O Vagrant dá para as equipes a possibilidade de garantir um ambiente virtual de desenvolvimento consistente e portátil que seja fácil e rápido de criar.
Então vamos começar as instalações!

Primeiramente começamos com o download do VirtualBox depois do Vagrant.

A instalação é default, seguindo a sequência do download.

Inicialmente, podemos utilizar uma BOX pré configurada com o Ubuntu pela Hashicorp.

Digitando o código vagrant init hashicorp/precise32 e depois vagrant up teremos umas maquina virtual rodando Ubuntu em poucos minutos!

Comandos mais utilizados:

Os comandos mais utilizados pelo vagrant são:

  • vagrant up
  • vagrant halt
  • vagrant box list
  • vagrant box remove *box*
  • vagrant box add *box*
